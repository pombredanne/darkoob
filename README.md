Installation
------------

Requirement
------------
Download and unzip the latest Elasticsearch distribution on http://www.elasticsearch.org/download/
Run bin/elasticsearch -f on Unix,
or bin/elasticsearch.bat on Windows
Run curl -X GET http://localhost:9200/
